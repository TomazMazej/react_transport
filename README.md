# React Transport
React app for getting a local transport.

## Usage and functions
* User registration and login
* Add/Remove/Edit transports  
* Transport reservation
* Rate customer/provider 
* Search/filter transports

## Tehnologies
* MongoDB
* Express
* React
* Node.js

## Start server
* cd ./server
* npm start

## Start Client
* cd ./client
* npm start
